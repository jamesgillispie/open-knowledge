# Pliable

Pliable is an analytics vendor whose product is an LLM chatbot ("Ellie") governed by a per-client semantic layer built entirely inside [Snowflake](/clients/snowflake/overview.md). Because the semantic layer lives in Snowflake, no client data egresses to an external LLM.

## Why it matters

Pliable claims 99.9% repeatable accuracy and a 2-3 day onboarding window. Its output is not locked to one surface: the governed query results can power Looker, OpenAI, Claude, or custom apps. It was evaluated for AREA 17 agency analytics on a joint call with [Snowflake](/clients/snowflake/overview.md).

The key differentiator is governance plus repeatability — the semantic layer constrains the LLM to deterministic, auditable answers, addressing the trust gap that usually blocks LLM-driven analytics.
