# Weekly Recap — Jun 1–5, 2026

Drawn from 19 Granola calls this week.

## Winston Taylor

The rest of the week was post-launch cleanup: DNS fixed (old Netlify A record deleted, wildcard CNAME added), Cloudflare cache rules set up after a CPU spike incident, staging.winstontaylor.com configured, and a OneTrust/GTM cookie bug resolved. Final bug list covers pixelated author photos, mobile header issues, people-page filter scroll, and preferred-name logic on PDFs. Project transitions to the Care team Monday with Clara as primary contact; first monthly report targeted for July 1.
